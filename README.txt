Para esta tarea se uso la aplicación XAMPP, que proporciona el acceso a los servidores.
Instrucciones:
Usamos las extensiones MySQL de Database Client (https://marketplace.visualstudio.com/items?itemName=cweijan.vscode-mysql-client2), PHP Intelephense (https://marketplace.visualstudio.com/items?itemName=bmewburn.vscode-intelephense-client) y PHP Server(https://marketplace.visualstudio.com/items?itemName=brapifra.phpserver) para poder visualizar y hacer funcionar el login y la navegación por las páginas, además, es necesario añadir a path la carpeta de PHP que se instaló durante el setup del xampp ("C:\xampp\php"), con eso hecho, se puede comprobar en la terminal de VScode si el php está instalado con 'php -v'. 
Con todo instalado, y habiendo activado los modulos Apache y MySQL en xampp, se puede ahora hacer clic derecho en cualquier parte del texto de index.php dentro de VScode, y hacer clic en PHP Server: Serve project para ingresar a la interfaz de inicio de sesión.
